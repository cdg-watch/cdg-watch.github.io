# アフィリエイト運用メモ

CDG Watch のマネタイズはサイト側のみで行う(**Xにはアフィリエイトリンクを貼らない**。
SNS.md の編集方針と信頼を守るため)。市場調査の結論(2026-07-09): X広告収益分配は
条件が遠く、当面はアフィリエイトが唯一の現実的な収益源。ただし期待値は
月数百円〜数千円規模なので、収益は「スコア表示」くらいの気持ちで見る。

## 現在の設置場所

| 場所 | 内容 |
|---|---|
| `index.html` の二次流通タブ | 楽天市場のコムデギャルソン検索への導線(`RAKUTEN_CDG` 定数) |
| `guide.html` | 定番アイテム購入ガイド。カテゴリ別の楽天検索リンク5本(`AFF_ID` 定数+`data-q`属性) |

- 楽天アフィリエイトの測定ID: `558d8f1f.1e88ee8e`(両ファイルで共通。変える時は両方)
- リンク形式: `https://hb.afl.rakuten.co.jp/hgc/<ID>/?pc=<encoded URL>&m=<encoded URL>`
- 表記ルール: アフィリエイトリンクには必ず `PR` チップと `rel="sponsored nofollow"` を付け、
  フッターに利用開示を書く(景表法のステマ規制対応。既存の文面を踏襲)

## ガイドページの育て方(エージェント向け)

- `guide.html` は検索流入(「コムデギャルソン 財布」等の購買意図クエリ)を狙う資産ページ。
  ニュース面と違い、内容は安定していてよい。手を入れるのは:
  - 価格目安が実勢から大きくズレたとき(年1回見直せば十分)
  - 定番に大きな動きがあったとき(コラボ終了・新定番の登場など、ニュースで把握したら)
- 効果測定は GoatCounter(cdg-watch.goatcounter.com)のページ別PVと、
  楽天アフィリエイト管理画面(snowが確認)のクリック/成果で行う。

## 今後の拡張(条件が揃ったら)

- **Amazonアソシエイト**: ファッション料率8%・報酬上限なしで楽天より単価が高い。
  ただし登録後180日以内に3成果が必要なので、**サイトが月数千PVに達してから**申請する
  (今申請すると審査落ちのリスクだけ負う)。承認されたら guide.html の各アイテムに
  Amazonリンクを併記する。
- ガイドページの追加候補: 「コムデギャルソンのライン一覧と見分け方」
  (検索需要が安定してあり、ニュースサイトとの相性も良い)
